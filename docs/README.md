# VerseDeck Companion Docs

Web estatica de documentacion para VerseDeck Companion.

## Vista local

Abre `index.html` directamente en el navegador o sirve esta carpeta con cualquier servidor estatico.

## Deploy en Vercel

1. Sube el repo a GitHub.
2. Crea un proyecto en Vercel.
3. En `Root Directory`, selecciona `docs`.
4. Deja `Build Command` vacio.
5. Deja `Output Directory` vacio.

Vercel servira `index.html` como pagina principal.
