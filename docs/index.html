Inicio / Introduccion

VerseDeck Companion

VerseDeck Companion es una botonera fan para Star Citizen centrada en una idea simple: ejecutar acciones manuales, configurables y locales desde escritorio, voz o movil en LAN.

Solo manual. Cada boton, frase de voz o toque movil dispara una sola pulsacion configurada por el usuario.

Inicio / Instalacion

Instalacion

Instala siempre la version mas reciente del instalador. Las actualizaciones se instalan encima de la version anterior y conservan tus perfiles, comandos y configuracion.

1

Ejecuta el instalador

Abre VerseDeckCompanionSetup-x.x.x.exe. La instalacion es por usuario y no necesita administrador.

2

Abre la app

Inicia VerseDeck Companion desde el menu de inicio o el acceso directo si decidiste crearlo.

3

Configura tu primer perfil

Selecciona una nave, revisa las teclas del MFD y guarda tu perfil.

Los datos del usuario viven en %AppData%\VerseDeck Companion. No se guardan dentro de la carpeta de instalacion.

Inicio / Primer arranque

Primer arranque recomendado

  1. Activa Modo editar antes de tocar el MFD si vas a configurar teclas.
  2. Selecciona un modulo, revisa Tecla y Modificadores.
  3. Pulsa Guardar modulo.
  4. Guarda el perfil con nombre y nave.
  5. Activa voz o servidor movil solo cuando ya tengas el deck basico preparado.

Perfil comandante

Un perfil es una configuracion completa para una nave o estilo de juego. Cada perfil tiene sus propios modulos, teclas y frases de voz.

Guia de usuario / MFD principal

MFD principal

El MFD contiene los botones de accion. Si Modo editar esta apagado, un click ejecuta la tecla. Si esta encendido, el click solo selecciona el modulo para editarlo.

Para configurar teclas sin enviar acciones accidentalmente, activa siempre Modo editar.

Guia de usuario / Editor del modulo

Editor del modulo

El editor controla el modulo seleccionado. Tambien permite crear y eliminar acciones personalizadas.

Identidad visual

  • Nombre del modulo: texto que aparece en el MFD.
  • Categoria: agrupa y ordena el modulo.
  • Icono: simbolo interno del boton.
  • Recuadro: cyan, verde o rojo.

Entrada

  • Tecla: tecla principal que se enviara.
  • Modificadores: Ctrl, Shift o Alt separados por coma.
  • Confirmacion secundaria: pide confirmar antes de ejecutar.

Acciones

  • Crear modulo nuevo: anade un boton al perfil actual.
  • Guardar modulo: guarda cambios del modulo seleccionado.
  • Eliminar modulo: borra el modulo y sus frases de voz.

Guia de usuario / Voz PTT

Voz PTT

La voz funciona con el reconocimiento local de Windows. VerseDeck prioriza es-ES y tambien puede usar reconocedores en ingles si estan instalados.

Anadir una frase de voz

  1. Selecciona un modulo del MFD.
  2. Escribe la frase en el campo de voz.
  3. Elige una confianza minima. Recomendado: 0.40 a 0.60.
  4. Pulsa Anadir frase de voz.
  5. Si la voz ya estaba activa, VerseDeck refresca el motor automaticamente.

Configurar Push-to-Talk

  1. Selecciona PushToTalk.
  2. Pulsa Detectar boton PTT.
  3. Suelta todo y pulsa teclado, raton, mando o joystick.
  4. Pulsa Guardar voz / PTT / Audio.
  5. Pulsa Activar voz.

Guia de usuario / Enlace movil

Enlace movil

El panel movil convierte cualquier telefono de la misma LAN en una botonera remota. La app muestra una URL y un QR para abrir el panel.

1

Activa el servidor

Pulsa Activar servidor en Enlace movil.

2

Escanea el QR

Abre el enlace desde el movil conectado a la misma red.

3

Toca una accion

El movil envia la misma entrada que el boton del MFD.

El movil usa WebSocket y respaldo HTTP. Si el socket no abre, los botones intentan enviar igualmente por HTTP.

Guia de usuario / Audio

Audio

Puedes activar sonido de bienvenida al iniciar y sonido system al ejecutar comando. Ambos son opcionales y nunca bloquean el envio de entrada.

Referencia / Como funciona

Como funciona

Accion humana
  -> Click en MFD / frase de voz / toque movil
  -> Perfil activo
  -> Tecla configurada del modulo
  -> Windows SendInput
  -> Star Citizen recibe una pulsacion normal

Referencia / Reglas seguras

Reglas seguras de entrada

No lee memoria del juego.

No intercepta red de Star Citizen.

No usa hooks del juego.

No hace loops ni macros.

No automatiza decisiones de gameplay.

Cada accion requiere una pulsacion, voz o toque humano.

Referencia / Problemas comunes

Problemas comunes

La app muestra un error o se cierra

Revisa %AppData%\VerseDeck Companion\crash.log y debug.log.

El movil carga botones pero no ejecuta

Comprueba que el telefono este en la misma LAN y revisa el estado mostrado arriba en el panel movil.

La voz no reconoce una frase

Usa frases cortas, baja temporalmente la confianza minima y confirma que Windows tenga reconocimiento es-ES.

Una tecla no funciona

Usa letras, numeros, F1-F12, flechas o teclas comunes soportadas. Evita simbolos no mapeados.
